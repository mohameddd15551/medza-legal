Privacy Policy

Last updated: May 20, 2026

Medza is a personal content-automation project operated by an individual creator ("the Operator"). This Privacy Policy describes how data is handled by Medza tools.

1. Whose Data We Handle

Medza handles only the Operator's own account data. The tools are not used by third parties, do not collect data from end users, and do not have signups, registrations, or public APIs.

2. What Data Is Accessed

3. Where Data Is Stored

All data — including OAuth tokens, video files, and configuration — is stored exclusively on the Operator's local machine. Nothing is transmitted to any Medza-controlled server because no such server exists.

4. What Data Is Shared

Data is shared only with the destination platforms during normal API operations:

  • TikTok (Content Posting API) — receives the uploaded video and post metadata;
  • Meta (Graph API for Instagram) — receives the uploaded video and caption;
  • YouTube (Data API) — receives the uploaded video, title, description, and thumbnail.

No data is sold, traded, given to advertisers, analytics services, or any other third party.

5. Token Security

OAuth access tokens are stored as local JSON files with restricted file-system permissions. Tokens are refreshed automatically before expiry using each platform's standard refresh-token flow. Revoking access at any time is possible by signing into the destination platform's app management page and removing the connected app.

6. Data Retention

Local data is retained only as long as the Operator chooses. Deleting the local files removes all stored data.

7. Children's Privacy

Medza tools are not directed at children. No data from minors is collected because the tools are not used by anyone other than the Operator.

8. Changes

If this policy changes, the "Last updated" date at the top will reflect the new revision.
